# Phase 7: Defense Evasion

## Overview
Defense Evasion involves techniques used to avoid detection by security tools, defenders, and security controls. This phase is critical for maintaining long-term access.

## Objectives
- Bypass security tools
- Avoid detection
- Remove indicators of compromise
- Blend in with normal activity
- Evade behavioral analysis

## Techniques

### Disable Security Tools
- Disable antivirus
- Disable EDR
- Disable firewalls
- Disable logging

### Impair Defenses
- Modify security configurations
- Disable security services
- Modify security policies

### Indicator Removal
- Clear logs
- Delete files
- Remove artifacts
- Cover tracks

### Obfuscated Files or Information
- Code obfuscation
- Encrypted payloads
- Packed executables

### Process Injection
- DLL injection
- Process hollowing
- Thread hijacking

## Tools
- Process Hacker
- Unhook tools
- Obfuscators
- Living off the land techniques

## Defensive Measures
- Behavioral analysis
- EDR solutions
- Log monitoring
- Anomaly detection
- Threat hunting
