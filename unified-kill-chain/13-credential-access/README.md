# Phase 13: Credential Access

## Overview
Credential Access involves obtaining credentials for authentication. This phase focuses on harvesting, stealing, and cracking credentials to gain access to systems and accounts.

## Objectives
- Obtain user credentials
- Extract password hashes
- Steal authentication tokens
- Crack passwords
- Harvest credentials from various sources

## Common Techniques

### OS Credential Dumping
- SAM database extraction
- LSASS memory dumping
- Credential manager access
- Browser password extraction

### Brute Force
- Password spraying
- Credential stuffing
- Dictionary attacks
- Online brute force

### Forced Authentication
- LLMNR/NBT-NS poisoning
- SMB relay attacks
- Forced authentication requests

### Keylogging
- Software keyloggers
- Hardware keyloggers
- Input capture

### Steal Application Access Token
- Browser tokens
- Application tokens
- API tokens
- OAuth tokens

## Tools
- Mimikatz
- LaZagne
- Hashcat
- John the Ripper
- BloodHound

## Defensive Measures
- Strong password policies
- Multi-factor authentication
- Credential protection
- Monitoring credential access
- Regular password audits
