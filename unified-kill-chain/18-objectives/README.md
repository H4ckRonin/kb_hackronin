# Phase 18: Objectives

## Overview
Objectives represent the socio-technical objectives of an attack that are intended to achieve a strategic goal. This phase focuses on understanding and achieving the attacker's primary motivations and strategic goals.

## Objectives
- Achieve primary goals
- Validate success
- Measure impact
- Document achievements
- Understand strategic goals

## Common Objectives

- Ransomware attacks
- Cryptocurrency mining
- Financial fraud
- Theft of financial data
- Payment card fraud

- Research data
