# Phase 4: Social Engineering

## Overview
Social Engineering involves manipulating humans to perform actions or divulge information that aids an attack. This phase focuses on exploiting human psychology rather than technical vulnerabilities.

## Objectives
- Manipulate targets
- Gain information
- Bypass technical controls
- Establish trust
- Achieve objectives through human interaction

## Common Techniques

### Phishing
- Email phishing
- Spear phishing
- Whaling
- Clone phishing

### Spear Phishing
- Targeted attacks
- Personalization
- Research-based
- High success rate

### Vishing
- Voice phishing
- Phone-based attacks
- Caller ID spoofing
- Pretexting over phone

### Pretexting
- False identity
- Scenario creation
- Trust building
- Information gathering

### Baiting
- Physical media
- USB drops
- Malicious downloads
- Curiosity exploitation

### Quid Pro Quo
- Service for information
- Tech support scams
- False assistance
- Information exchange

### Physical Social Engineering
- Tailgating
- Shoulder surfing
- Dumpster diving
- Impersonation

## Tools
- Gophish
- King Phisher
- SET (Social Engineering Toolkit)
- Custom templates

## Defensive Measures
- Security awareness training
- Email filtering
- Multi-factor authentication
- Verification procedures
- Incident response
