# Phase 16: Exfiltration

## Overview
Exfiltration involves techniques that result or aid in an attacker removing data from a target network. This phase focuses on securely transferring collected data to attacker-controlled infrastructure.

## Objectives
- Transfer data securely
- Evade detection
- Handle large datasets
- Maintain data integrity
- Use multiple channels

## Common Techniques

### Exfiltration Over C2 Channel
- Using existing C2 infrastructure
- Encrypted transfers

### Exfiltration Over Alternative Protocol
- FTP
- SCP
- Custom protocols

### Exfiltration Over Web Service
- Cloud storage
- Web services
- Paste sites

### Scheduled Transfer
- Automated transfers
- Scheduled exfiltration

### Data Transfer Size Limits
- Chunking large files
- Compression
- Encryption

## Tools
- Custom scripts
- C2 frameworks
- Cloud storage APIs
- Web services

## Defensive Measures
- Network monitoring
- Data loss prevention (DLP)
- Egress filtering
- Anomaly detection
- Data classification
