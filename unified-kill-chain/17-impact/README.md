# Phase 17: Impact

## Overview
Impact involves techniques aimed at manipulating, interrupting or destroying the target system or data. This phase focuses on actions that directly affect the target's operations, data, or systems.

## Objectives
- Disrupt operations
- Destroy data
- Encrypt data
- Modify systems
- Demonstrate impact
- Cause service degradation

## Common Techniques

### Data Encrypted for Impact
- Ransomware
- Data encryption
- File encryption

### Data Destruction
- File deletion
- Disk wiping
- Database corruption
- Backup deletion

### Defacement
- Website defacement
- System message modification
- Brand damage

### Service Stop
- Service termination
- Process termination
- System shutdown
- Service disruption

### Account Access Removal
- Account deletion
- Password changes
- Access revocation

### Inhibit System Recovery
- Backup deletion
- Recovery partition removal
- System restore disable

### Resource Hijacking
- Cryptocurrency mining
- Compute resource theft
- Bandwidth consumption

## Tools
- Custom scripts
- System commands
- Ransomware tools
- Wipers

## Defensive Measures
- Backups
- Redundancy
- Monitoring
- Incident response
- Business continuity planning
