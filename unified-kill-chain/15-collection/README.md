# Phase 15: Collection

## Overview
Collection involves techniques used to identify and gather data from a target network prior to exfiltration. This phase focuses on locating and collecting valuable information from compromised systems.

## Objectives
- Identify valuable data
- Collect credentials
- Gather sensitive information
- Capture user input
- Document findings
- Organize collected data

## Common Techniques

### Data from Local System
- File system enumeration
- Registry data
- Configuration files
- Application data

### Data from Network Shared Drive
- SMB shares
- NFS shares
- Cloud storage
- Network file shares

### Data from Information Repositories
- Databases
- Document management systems
- Code repositories
- Email systems

### Screen Capture
- Screenshot tools
- Screen recording

### Clipboard Data
- Clipboard monitoring
- Clipboard capture

### Keylogging
- Software keyloggers
- Hardware keyloggers

## Tools
- Mimikatz
- LaZagne
- BloodHound
- Custom collectors
- Screen capture tools

## Defensive Measures
- Data classification
- Access controls
- Encryption
- Monitoring data access
- DLP solutions
