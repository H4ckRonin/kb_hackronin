## Overview

Active Directory certifications focus on:
- Active Directory enumeration and reconnaissance
- Privilege escalation (local and domain)
- Lateral movement techniques
- Persistence mechanisms
- Cross-trust attacks
- Defense bypasses (MDE, MDI, Windows Defender)
- Azure AD and cloud-based Active Directory
