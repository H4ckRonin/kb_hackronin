# eCPPTv2 - eLearnSecurity Certified Professional Penetration Tester v2

## Overview
The eLearnSecurity Certified Professional Penetration Tester v2 (eCPPTv2) is an intermediate-level penetration testing certification from INE Security (formerly eLearnSecurity).

**Source**: [johnermac.github.io](https://johnermac.github.io/notes/ecppt/)

## Exam Details
- **Provider**: INE Security (formerly eLearnSecurity)
- **Focus**: Advanced penetration testing methodologies
- **Format**: Practical hands-on exam with report
- **Difficulty**: Intermediate
- **Duration**: 7 days
