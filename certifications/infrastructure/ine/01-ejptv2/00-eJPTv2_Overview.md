# eJPTv2 - eLearnSecurity Junior Penetration Tester v2

## Overview
The eLearnSecurity Junior Penetration Tester v2 (eJPTv2) is an entry-level penetration testing certification from INE Security (formerly eLearnSecurity).

**Source**: [johnermac.github.io](https://johnermac.github.io/notes/ejpt/)

## Exam Details
- **Provider**: INE Security (formerly eLearnSecurity)
- **Focus**: Foundational penetration testing skills
- **Format**: Practical hands-on exam
- **Difficulty**: Entry-level
- **Duration**: 3 days

## Key Topics

- Information Gathering
- Scanning and Enumeration
- Vulnerability Assessment
- Web Application Security
- System Security
- Network Security
- Exploitation Basics
- Report Writing
