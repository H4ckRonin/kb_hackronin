# CompTIA Certifications

CompTIA (Computing Technology Industry Association) provides vendor-neutral IT certifications covering various cybersecurity domains.

## Certifications

- [01 - PenTest+ ce - Penetration Testing+](01-pentest-plus/00-pentest-plus_Overview.md)
  - Intermediate-level penetration testing certification
  - Validates skills in planning, conducting, and reporting penetration tests

- [02 - Security+ ce - Security+](02-security-plus/00-security-plus_Overview.md)
  - Foundational cybersecurity certification
  - Validates baseline security skills
