# eMAPTv3 - Mobile Application Penetration Tester v3

## Overview
The INE Security's Mobile Application Penetration Tester (eMAPTv3) certification focuses on mobile application security testing for both iOS and Android platforms.

**Official Site**: [INE Security eMAPT](https://security.ine.com/certifications/emapt-certification/)

## Exam Details
- **Provider**: INE Security (formerly eLearnSecurity)
- **Focus**: Mobile application penetration testing
- **Format**: Practical hands-on exam
- **Time Limit**: TBD
- **Difficulty**: Intermediate to Advanced

## Topics Covered
- Mobile application security fundamentals
- iOS application security
- Android application security
- Mobile application reverse engineering
- Mobile application exploitation
- Mobile application security assessment methodology
