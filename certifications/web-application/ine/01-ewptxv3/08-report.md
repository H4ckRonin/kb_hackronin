# Report Writing - eWPTXv3

## Overview
How to write professional penetration testing reports.

**Source**: [dev-angelist GitBook](https://dev-angelist.gitbook.io/ewptxv3-notes/report)

## How to Write a PT Report

### Report Structure
1. **Executive Summary**
   - Overview
   - Key findings
   - Risk summary
   - Recommendations

2. **Introduction**
   - Scope
   - Objectives
   - Methodology
   - Limitations

3. **Findings**
   - Vulnerability details
   - Proof of concept
   - Impact assessment
   - Risk rating

4. **Recommendations**
   - Remediation steps
   - Best practices
   - Priority guidance

5. **Appendices**
   - Tools used
   - References
   - Screenshots
   - Raw data

### Key Elements
- Clear and concise language
- Technical accuracy
- Risk-based prioritization
- Actionable recommendations
- Professional presentation

## Tools
- **Dradis** - Report generation
- **Serpico** - Report templates
- **Microsoft Word/LaTeX** - Document creation
