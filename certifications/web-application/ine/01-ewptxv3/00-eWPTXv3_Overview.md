# eWPTXv3 - Web Application Penetration Tester eXtreme v3

## Overview
The INE Security's Web Application Penetration Tester eXtreme (eWPTXv3) certification is a hands-on exam designed for cybersecurity professionals with intermediate to advanced expertise in web application security and penetration testing.

**Source**: [dev-angelist GitBook](https://dev-angelist.gitbook.io/ewptxv3-notes)

**Official Site**: [INE Security eWPTX](https://security.ine.com/certifications/ewptx-certification/)

## Exam Details
- **Provider**: INE Security (formerly eLearnSecurity)
- **Focus**: Advanced web application penetration testing
- **Format**: Multiple-choice quiz (through lab environment)
- **Time Limit**: 18 hours
- **Expiration**: 3 years
- **Difficulty**: Intermediate to Advanced

## Course Duration & Topics
