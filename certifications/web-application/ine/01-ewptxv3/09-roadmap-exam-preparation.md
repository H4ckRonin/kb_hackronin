# RoadMap / Exam Preparation - eWPTXv3

## Overview
Study roadmap and exam preparation guide for eWPTXv3.

**Source**: [dev-angelist GitBook](https://dev-angelist.gitbook.io/ewptxv3-notes/roadmap-exam-preparation)

## Exam Details
- **Type**: Multiple-choice quiz (through lab environment)
- **Time Limit**: 18 hours
- **Expiration**: 3 years

## Study Roadmap

### Phase 1: Foundation
- Web application basics
- HTTP/HTTPS protocols
- OWASP Top 10
- Basic tools (Burp Suite, OWASP ZAP)

### Phase 2: Core Topics
- Authentication mechanisms
- Session management
- Injection vulnerabilities
- XSS and CSRF

### Phase 3: Advanced Topics
- API testing
- Server-side attacks
- WAF bypass
- Advanced exploitation

### Phase 4: Practice
- Lab environments
- Practice applications
- PortSwigger Web Security Academy
- DVWA, SecureBank, BWAPP

## Practice Labs
- Xampp and Mutillidae II
- PortSwigger Web Security Academy
- DVWA (Damn Vulnerable Web Application)
- SecureBank
- BWAPP (Buggy Web Application)

## Exam Tips
- Read questions carefully
- Test in lab environment
- Document findings
- Manage time effectively
- Review all topics
