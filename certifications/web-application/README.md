## Overview

Web application certifications focus on:
- Web application reconnaissance
- Injection vulnerabilities (SQL, XSS, Command, etc.)
- Authentication and session management attacks
- API penetration testing
- Filter evasion and WAF bypass
- Server-side attacks
- Web application methodology
- Reporting and documentation
